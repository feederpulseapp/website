Changelog & Releases

Stay up to date with the latest features, improvements, and bug fixes for FeederPulse.

v1.3.0

May 30, 2026

Added

  • Session Pulse Analysis: A premium, ECG-inspired curved line chart displaying raw scores, momentum adjustments, and smoothing values reactively.
  • Biometric Pulse Metrics: Dynamic calculation of Peak/Average Pulse scores, standard deviation, and session stability.
  • Smart Period Detection: Automated, sliding 40-minute "Best Window" calculation and low-activity "Dead Zones" detection.
  • Automated Bank Insights: Textual summaries detailing catch pace, lulls, momentum spikes, and trailing fatigue.
  • History Pulse Sparklines: Mini visual pulse fingerprints embedded inside finished session list items.

Changed

  • Settings UI Optimization: Repositioned the app version dynamic label on the settings screen for a cleaner, more intuitive bottom layout.

v1.2.0

May 29, 2026

Added

  • Team-based Competitions: You can now mark a competition as team-based during setup. Team-based competitions are visually distinguished in the session list with a team icon.
  • Team Ranking Visualization: The team icon in the session list dynamically colors to Gold, Silver, or Bronze based on your final team placement.
  • Final Weigh-in Feature: Added a dedicated flow for recording the final weigh-in of all keepnets at the end of a session, locking the individual catch editing to ensure data integrity.

Changed

  • Individual Ranking Editing: Redesigned the ranking edit experience. Instead of a single modal for all rankings, you can now tap directly on Sector, Overall, or Team to edit them independently.
  • Improved Statistics Accuracy: Virtual direct weigh-ins (fishes weighed without being assigned to a keepnet) are now accurately aggregated into the final session statistics.
  • Localization Updates: Improved translation accuracy.

v1.1.3 (Build 5)

May 25, 2026

This milestone release introduces a fully customizable, drag-and-drop Active Session Layout for advanced tracking, alongside a beautifully simplified Casual (Everyday Leisure) Mode.

Casual Mode (Fishing Time)
Casual Mode (Cast Time)

Key Enhancements

  • Customizable Active Session Layout (Beta): Take complete control of your bank-side active workspace with interactive, reorderable widgets.
  • Casual / Leisure Active Session Mode: Log leisure fishing sessions with a beautiful, green-themed interface completely stripped of competitive elements.
  • Integrated Dashboard & 3D Flip Timer: Reduced the circular timer frame and embedded it inside the presets control panel with a premium 3D flip interaction.
  • Smart Distance Presets & Premium Bottom Sheets: Seamless cache reusability and modernized dialog layouts.
  • Global Timer Mode: Added a new setting to toggle between Count Up and Countdown timer modes globally.
  • Practice End Times: Added support for an optional end time for Quick Practice sessions.

Fixed & Polished

  • Session End Notification & Timer Teardown: Fixed an alert leak where ending an active session early left background cast notifications active.
  • Background Opacity Fix: Corrected catch detail sheets to render with solid surface colors.
  • Distance Chart Resiliency: Fixed an issue in the Catch Distance Chart.

v1.1.2

May 20, 2026

Added

  • Casual Session Mode: Introduced a new session type for casual fishing without strict competition rules.
  • Customizable Active Session Layout: Enhanced the active session screen with toggle options to hide/show specific elements and reorder them.
  • Distance Analytics: Added a new "Species by Distance" chart and refined catch list styling.

v1.1.0

May 15, 2026

Added

  • Session Detail Statistics: Added detailed KPIs including catch rate (fishes/hr), weight rate (grams/hr), and catch per cast ratio.
  • Extreme Sun Theme: Added a high-contrast Extreme Sun theme for better visibility during bright days on the bank.
  • Quick Scheduling: Introduced quick-scheduling features for upcoming sessions with push reminders and organizer notifications.
  • Live Stats Toggle: Added a setting to turn live statistics calculation on/off during active sessions.

v1.0.0

Initial Release

Features

  • Core Competition Engine: Full support for tracking catches, keepnet limits, casts, and overall weights during matches.
  • Offline-First Data: All session records and catch logs are stored securely on your device, so you can track your fishing anywhere on the bank without an internet connection.
  • Dark Mode First UI: Battery-saving dark interfaces built for sunlight readability and one-handed operation.
  • Localization: Full Hungarian and English language support.
